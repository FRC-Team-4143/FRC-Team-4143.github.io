# Fusion Guide

The following text will walk you through how to set up and generate G-Code from Fusion.
If you do not have access to Fusion or the shared work space refer to [**Fusion Sign-Up**](https://frc-team-4143.github.io/docs/mechanical/fusion/fusion-sign-up)

## Exporting from Onshape to Fusion 360

- Right click the part in Onshape model or the part tree and click **Export**.

<img src={onshape_export} width="1000"/>

- Confirm the part has a descriptive name.
    - Subassembly
    - Part Name
    - Date of Export
- Format: Step
- Option: Download
<img src={onshape_download} width="400"/>

- Navigate to Autodesk Fusion 360.
- Open the Data Panel in the upper left corner.
- Select the appropriate project folder (2026FRC4143).
<img src={Upload_1} width="400"/>

- Navigate to the desired subassembly folder.
<img src={Upload_2} width="400"/>


- Click the Upload button in the Data Panel.
- Drag and drop the exported Step file into the upload window.
- Once file is loaded Click Upload.
- Once the file is uploaded, double click the part to open it in Fusion.
<img src={Upload_3} width="400"/>
