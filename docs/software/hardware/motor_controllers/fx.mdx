The CTRE Talon FX is a high-performance brushless motor controller commonly used in our robots. It offers percent output,
velocity, position, and torque control modes. The Talon FX also includes advanced features such as integrated PID control,
and customizable CAN bus communication.

<img src={talon_fx} width="400"/>

Here is a link to the CTRE API documentation for the Talon FX: [Talon FX API](https://v6.docs.ctr-electronics.com/en/latest/docs/hardware-reference/talonfx/index.html)
