# RoboRio
The RoboRio is the main compute system used in FRC robots. It is responsible for running the robot code, managing communication between various subsystems, and interfacing with sensors and actuators on the robot. The RoboRio is a embedded system that provides a robust platform for developing and deploying robot software.
