# Software Docs

Welcome to the software documentation for the MARS/WARS robotics team! This documentation covers everything you need to know about programming and software development for our robots.

## What You'll Find Here

**Software Tools** - Essential development tools including Git, VS Code, and robot-specific software

**Java Basics** - Fundamental Java programming concepts including variables, classes, loops, and more

**Robot Development** - WPILib framework, subsystems, commands, and robot programming patterns

**Advanced Controls** - Control theory, PID controllers, and advanced robot control concepts  

**Robot Communications** - Networking, protocols, and communication between robot components

**Lessons Learned** - Best practices and insights from our development experience

Navigate through the sections using the sidebar to explore specific topics and tutorials.
