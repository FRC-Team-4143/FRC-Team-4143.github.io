# Advantage Scope
Advantage Scope is an advanced telemetry and log viewing tool developed by FRC Team 6328. They have excellent documentation available [here](https://docs.advantagescope.org/). This page should act only as a quick reference for common actions and you should check their official documentation for more in depth instructions.
